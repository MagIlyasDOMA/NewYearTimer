# Таймер до Нового года
## Как использовать:

#### 1. Установите Node.js

#### 2. Клонируйте репозиторий
```shell
git clone https://github.com/MagIlyasDOMA/NewYearTimer.git
```

#### 3. Перейдите в директорию
```shell
cd NewYearTimer
```

#### 4. Установите зависимости
```shell
npm install
```

#### 5. Запустите сервер
```shell
npm run runserver
```
