# Таймер до Нового года
## Как использовать:

#### 1. Установите Node.js

#### 2. Клонируйте репозиторий
```shell
git clone https://github.com/MagIlyasDOMA/NewYearTimer.git
```

#### 3. Перейдите в директорию
```shell
cd NewYearTimer
```

#### 4. Установите зависимости
```shell
npm install
```

#### 5. Задайте конфигурацию таймера
```json
{
  "year": 2026, // Год
  "sound": true, // Включить звук
  "soundIfIsHidden": false, // Оставлять звук, если вкладка выключена
  "timerTransparency": 8 // Прозрачность фона таймера (0-10)
}
```

#### 6. Запустите сервер
```shell
npm run runserver
```
